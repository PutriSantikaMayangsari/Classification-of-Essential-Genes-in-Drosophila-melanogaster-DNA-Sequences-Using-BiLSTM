Project Status: Completed Bachelor's Thesis

Objective:
Classification of essential genes in Drosophila melanogaster DNA sequences.

Method:
- Bidirectional LSTM

Dataset:
- Essential and non-essential gene sequences from Drosophila melanogaster.
- Dataset derived from the study:
  "Identifying Essential Genes Across Eukaryotes by Machine Learning"
  (Beder et al., NAR Genomics and Bioinformatics, 2021).

Note:
Source code is not publicly available.
